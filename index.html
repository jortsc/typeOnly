The jQuery plugin to type only specifics characters in a given input text element.

Allows to control to the characters that will be typed in an input field. By defaults only numbers are allowed. But you can allow decimal numbers with a custom decimal separator there are two allowed, or only alphabetical characters also. All they with a given length which is passed as a parameter.

Default options:
type : 'num'
digits : 8
separator :'.'

The parameter called 'type' which determine the characters will be typed.

Its allowed values are:
num only numbers
dec decimal number
alpha alphabetical characters. These are limited, you are free of add some more.

The 'digits' parameter.

digits are the length of the value, is an integer.

The 'separator' parameter.

separator is the separator for the decimal number only can be '.' dot or ',' comma and only one.

Examples

$(selector).typeOnly(); $(selector).typeOnly({type:'dec',digits:12,separator:','});
$(selector).typeOnly({type:'alpha',digits:2});
